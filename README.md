# 👥 People Detection with YOLOv8

A computer vision project that detects people in images using the YOLOv8 object detection model. The model is trained on a custom dataset from Roboflow and demonstrates the complete workflow of dataset preparation, model training, and inference.

## Features

- Custom people detection model
- YOLOv8 implementation using Ultralytics
- Roboflow dataset integration
- Model training and evaluation
- Image inference and visualization
- Easy to customize for other object detection tasks

## Tech Stack

- Python
- YOLOv8 (Ultralytics)
- Roboflow
- OpenCV
- Matplotlib

## Project Workflow

1. Download the dataset from Roboflow.
2. Train a YOLOv8 model on the dataset.
3. Run inference on test images.
4. Visualize detected people with bounding boxes.
